# Map Assets

Place the Divine Right Third Edition map image here.

## Expected file

- `minaria-map.png` or `minaria-map.jpg` - The full DR3 Minaria map

## Calibration

Once the map image is added, use the calibration controls in the app to align the hex grid overlay with the map. The calibration settings (offsetX, offsetY, hexScale) can then be saved to `src/config/mapCalibration.ts`.
